      欢迎来到 Ode Mod 文档

      这里是有关开源模组 Ode Mod 的 API文档。

      概述

      Ode Mod 是基于游戏 Terraria 的模组加载器 tModLoader 的，由 Ode Team 开发的 Mod。Ode Mod 目前已于 GitHub 平台之上开放了全部源代码， 你可以通过访问 https://github.com/zqzcsilver/OdeMod 来查看它们，其遵循 MIT开源协议。本文档仅包含非 tModLoader 部分代码的 API文档 。
